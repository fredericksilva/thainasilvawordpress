Installation

  • Item: Kalium Getting Started

To install Kalium theme first you must have a working WordPress site, the theme works on any version of WordPress 4.1.x.x just make sure you follow the instructions step by step. If you have trouble installing WordPress read this tutorial.

There are two ways to install Kalium.

  1. Via WordPress – Log in into the admin area, navigate through Appearance > Themes > Install Themes > Upload, now make sure to unzip the file you downloaded and then and select the kalium.zip file from the downloaded files and finally click on Install Now
  2. Via FTP – The other way is by FTP, log in into your site’s FTP through a FTP Client and go into the WordPress installation directory and then navigate to:/wp-content/themes/ and upload the kalium folder here. After the uplad is complete go to  Appearance > Themes into your admin area, you will see Kalium theme there, hit Activate to activate Kalium.
