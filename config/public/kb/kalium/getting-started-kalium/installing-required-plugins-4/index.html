Installing Required Plugins

  • Item: Kalium Getting Started

This step is very important, because your theme functionality depends on these settings, so please follow this section carefully.

After you upload and install/activate Kalium theme this message will appear:

You should click Begin installing polugins and Install/Activate all required plugins:

 

The only optional plugin for this theme is Envato WordPress Toolkit which allows you to get updates for Kalium theme.

After you install and activate all the required plugins, then continue to the below section with “Theme Configuration“.
